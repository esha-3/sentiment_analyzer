# Sentiment Analyzer 

A beautiful, AI-powered web application that analyzes the emotional tone behind any text in real-time. Built as a student project, this app takes your text (tweets, reviews, articles) and tells you if it's positive, negative, or neutral!

##  Features

- **Real-time Sentiment Analysis**: Instantly get the emotional tone (positive, negative, or neutral) of your text.
- **Detailed Metrics**: See exact scores for *Polarity* (how positive/negative the text is) and *Subjectivity* (how opinionated or factual it is).
- **Beautiful Interface**: A modern, dark-themed UI featuring glassmorphism, animated backgrounds, and dynamic visual gauges.
- **Analysis History**: Keeps track of your recent analyses so you can easily compare different texts.

##  Built With

- **Backend**: Python & Django (Handles the API and logic)
- **Frontend**: HTML, CSS, and JavaScript 
- **NLP**: Natural Language Processing to analyze text polarity and subjectivity.

##  How to Run Locally

To run this project, follow these simple steps:

1. **Clone the repository:**
   ```

   cd your-repo-name
2. **Set up a Virtual Environment**
   python -m venv env
  # On Windows:
    env\Scripts\activate
  # On Mac/Linux:
    source env/bin/activate 
3. **Install the requirements:**
   pip install django 
4. **Run the server:**
   cd myproject
   python manage.py runserver 8001
5. **Open your browser:**
   Go to http://127.0.0.1:8001 and start analyzing text!
